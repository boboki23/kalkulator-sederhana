# Kalkulator Sederhana

Aplikasi kalkulator desktop sederhana berbasis **Python** dan **Tkinter**.

## Fitur
- Penjumlahan (+)
- Pengurangan (-)
- Perkalian (X)
- Pembagian (/)
- Modulo/Sisa Bagi (%)
- Mendukung angka desimal (float) dan bilangan bulat (integer)
- Keyboard support (angka, operator, Enter, Backspace, ESC)
- Batasan maksimal 15 digit

## Cara Menjalankan
1. Pastikan Python sudah terinstall di komputer Anda.
2. Buka terminal/command prompt di folder Anda
3. Jalankan perintah:
   ```bash/cmd/terminal
   python kalkulator.py
